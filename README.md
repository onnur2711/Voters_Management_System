 Voter Management System

A Full Stack Web Application developed using *React.js, **Spring Boot, and **MySQL* for managing voter details efficiently.

This project performs complete CRUD operations with proper frontend and backend validations.

---

## Features

- ✅ Add New Voter
- ✅ View All Voters
- ✅ Read Voter By ID
- ✅ Search Voter By Aadhaar
- ✅ Search Voter By Mobile Number
- ✅ Update Voter Details
- ✅ Delete Voter
- ✅ Backend Validations
- ✅ Responsive User Interface
- ✅ REST API Integration
- ✅ MySQL Database Connectivity

## Frontend Setup
Bash
cd frontend
npm install
npm start
Frontend runs on:
Plain text
http://localhost:3000

## Backend Setup
Bash
cd backend
Run the Spring Boot application from Eclipse / STS.
Backend runs on:
Plain text
http://localhost:8800

## Database Configuration
Update application.properties with your MySQL credentials.
Example:
Properties
spring.datasource.url=jdbc:mysql://localhost:3306/votersdb
spring.datasource.username=root
spring.datasource.password=yourpassword
